# Site Profissional — Psicóloga Giovana Frassetto

Landing page profissional desenvolvida para apresentar os serviços, abordagem terapêutica, formas de atendimento e espaço clínico da psicóloga **Giovana Frassetto**.

O projeto foi construído com foco em uma experiência visual acolhedora, responsiva e elegante, priorizando boa navegação em dispositivos móveis e desktop.

---

## Sobre o projeto

Este site tem como objetivo apresentar de forma clara e sensível o trabalho clínico da profissional, criando uma presença digital humanizada e acessível.

A página conta com seções como:

* Hero inicial com chamada principal
* Menu responsivo com navegação suave
* Apresentação profissional
* Explicação sobre como a psicoterapia pode ajudar
* Informações sobre atendimento
* Galeria de fotos da clínica
* Botões de contato direto via WhatsApp

---

## Tecnologias utilizadas

O projeto foi desenvolvido com:

* **React**
* **TypeScript**
* **Vite**
* **Material UI**
* **Framer Motion**
* **MUI Icons**


## Identidade visual

A identidade do site utiliza uma paleta suave e acolhedora, com tons terrosos, fundos claros e contrastes delicados.

Principais características visuais:

* Layout limpo e elegante
* Bordas arredondadas
* Cards coloridos
* Degradês suaves
* Animações leves
* Imagens em destaque
* Tipografia com sensação acolhedora
